유튜브는 사용자의검색 이력과 시청 기록을 저장하여 추천 알고리즘을 최적화하는 기능을 제공합니다. 하지만개인정보 보호 및 맞춤 추천 기능을 원하지 않는 경우, 검색 기록을 끄거나 삭제할 수 있습니다.이번 가이드에서는유튜브 검색 이력을 제외하는 방법, 기록 삭제 방법, 완전히 끄는 설정 방법을 상세히 알려드립니다.


## 유튜브 검색 이력 기능 제외하는 방법


유튜브에서 검색 기록을 저장하지 않도록 설정하는 방법은 크게 두 가지입니다.

✅검색 기록 삭제– 기존 검색 이력을 개별 또는 전체 삭제

✅검색 기록 중지– 검색 기록이 저장되지 않도록 설정


### ? 검색 기록 삭제하는 방법 (PC & 모바일 공통)


1.유튜브 접속 후 로그인

•PC:[유튜브 웹사이트](https://www.youtube.com/)접속 후 우측 상단의프로필 아이콘 클릭

•모바일:유튜브 앱 실행 후 우측 상단프로필 아이콘 클릭

2.설정(Settings) 이동

•PC:좌측 메뉴에서‘기록’클릭

•모바일:프로필 아이콘 클릭 후‘설정’ → ‘기록 및 개인정보 보호’선택

3.검색 기록 삭제

•‘검색 기록 관리’ → ‘모든 검색 기록 삭제’클릭

•특정 검색어만 삭제하려면 검색 기록에서 개별 삭제 가능

?주의:검색 기록을 삭제하면삭제된 데이터는 복구할 수 없으므로 신중히 결정해야 합니다.


### ? 검색 기록 저장 중지하는 방법 (자동 저장 차단)


1.유튜브 로그인 후 기록 설정 이동

•[유튜브 기록 설정 페이지](https://myactivity.google.com/product/youtube?pli=1)접속 - 이 부분은 클릭을 통해 옳바로 이동하면 좋지만 그렇지 않다면 자체적으로 검색 후에 해결책을 찾아보세요. 개인 설정이다 보니 다를 수 있습니다.

•PC & 모바일 동일한 과정

2.검색 기록 저장 중지 설정

•‘유튜브 검색 기록’ 항목에서‘검색 기록을 일시 중지’버튼 클릭

•확인 메시지에서‘일시 중지’선택

3.검색 기록이 저장되지 않는지 확인

•이후 검색을 실행해도 기록이 남지 않는지 직접 확인 가능

?Tip:이 설정을 하면앞으로 유튜브 검색 기록이 남지 않으며, 추천 알고리즘에도 반영되지 않습니다.


### ? 자동 삭제 설정 (검색 기록을 일정 기간 후 자동 삭제)


1.[Google 활동 관리 페이지](https://myactivity.google.com/activitycontrols/youtube)이동

2.‘유튜브 검색 기록’ 클릭 후 ‘자동 삭제 설정’

3.삭제 주기 선택 (3개월 / 18개월 / 36개월 단위)

4.‘설정 완료’ 클릭하여 자동 삭제 적용

?Tip:검색 기록을 자동으로 삭제하는 기능을 활성화하면, 일정 기간 후 검색 기록이 자동으로 사라집니다.


### 유튜브 검색 기록 제외 설정의 장점 & 단점


✅장점:

•검색 기록을 남기지 않아 개인정보 보호 강화

•다른 사람이 내 유튜브 검색 기록을 볼 위험 차단

•추천 알고리즘에 영향을 주지 않아 새로운 콘텐츠 탐색 가능

❌단점:

•맞춤 추천 기능이 비활성화되어 유튜브가 선호하는 콘텐츠를 추천하지 않음

•검색 기록을 활용한 빠른 접근이 불가능

유튜브의 검색 이력 기능을 제외하는 방법은 간단합니다.

2️⃣검색 기록 저장 자체를 중지하거나,

3️⃣자동 삭제 설정을 적용하는 방법이 있습니다.
