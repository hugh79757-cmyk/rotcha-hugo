네이버 블로그 홈페이지형 만드는 과정을 유튜브와 블로그의 글을 보고 하나하나 따라 해 봤어요. 하지만 막히는 부분이 있었습니다. 공간 위젯을 만들어서 네이버에서 설정된 타이틀이나 메뉴를 아래로 밀어내서 배경 스킨이 보이게 만들어야 하는데, 저는 아무리 해도 배경 스킨을 가린 형태가 되더군요. 결국스킨 변경이정답이었습니다.

네이버 블로그 홈페이지형을 만들 때 순서 개념 잡기#

스킨 배경과 레이아웃 설정하기#

  • 2000*700(혹은 600)의 사이즈로 스킨 배경을 만들어 줍니다.
  • ‘관리-꾸미기 설정-세부 디자인 설정-디자인 직접 등록’을 통해 스킨 배경을 등록합니다.
  • ‘관리-꾸미기 설정-레이아웃.위젯 설정’에서 아래 사진과 같은 레이아웃을 선택합니다.

투명 위젯 설정하기#

  • ‘관리-꾸미기 설정-레이아웃.위젯 설정-위젯 직접 등록’을 통해서 투명 위젯을 5개 만들어줍니다.
  • 투명 위젯을 설정해서 ‘글영역’을 원하는 만큼 아래로 밀어내서 처음에 설정한 ‘스킨 배경’이 보이도록 해 주세요.

투명 위젯 만들기#

투명 위젯의 html 코드는 다음과 같습니다.

  1. 테이블을 이용한 방법

위의 코드를 이용해서 투명위젯을 직접 등록합니다. 높이를 자신의 이미지에 맞게 height=“600"에서 600의 숫자를 바꿔주시면 됩니다.

  1. 줄바꿈을 이용한 방법

이와 같은 코드를 연속으로 넣어줘서 투명 위젯의 길이를 조정합니다.

5개의 투명위젯 이름을 공간 1, 공간 2, 공간 3, 공간 4, 공간 5로 설정하고 그림처럼 ‘메뉴 형태’아래 배치해 줍니다. 이때 오른쪽 ‘레이아웃 설정’은 넓게를 선택해 주세요.

저는 여기서 문제가 발생했습니다.

투명위젯이 적용이 안 되는 문제 해결#

위의 그림과 같이 ‘글영역’이 투명 위젯을 적용한 이유로 아래로 내려가지만 뒤의 배경 이미지는 그대로 가려진 경우가 생겼습니다.

이때 ‘스킨’을 변경해 주면 해결이 됩니다.스킨의 종류에 따라서 위와 같은 문제가 발생하기도 합니다.

해결을 위해 저는 ‘관리-꾸미기 설정-스킨 선택’에서 두 번째 스킨인 ‘와이드’를 선택하고 적용시켰습니다.

스킨을 변경시킨 후 위의 사진과 같이 정상적으로 글 영역이 아래로 내려간 것을 확인할 수 있었습니다.

스킨 배경 위로 버튼을 추가하신다면 투명 위젯의 테이블 높이를 조정해서 버튼 1, 버튼 2 ~ 버튼 5까지 만든 다음 공간 1~공간 5 아래 배치시켜줍니다. 공간 위젯의 테이블 높이는 height=300, 버튼 위젯의 테이블 높이도 300으로 설정해주면 됩니다.

자세한 내용은 유튜브나 다른 분의 블로그를 참조하시기 바랍니다.

마무리#

네이버 블로그 홈페이지형을 만들 경우의 이점은 좀더 전문적인 블로그로 보일 수 있다는 점입니다. 획일화된 다른 분들의 블로그와 차별화된 나만의 고유한 블로그 레이아웃의 블로그를 원하신다면 시도하시기를 추천드립니다. ‘스킨 변경’을 모르는 상태로 여러 번 시행착오를 거치다 알게 돼서 포스팅합니다. 나만의 예쁜 네이버 블로그 홈페이지형을 만드시는데 참고하시기 바랍니다.
