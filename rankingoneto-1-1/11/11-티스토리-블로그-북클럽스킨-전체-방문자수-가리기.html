티스토리 블로그 북클럽스킨 전체 방문자수 가리기

지식 아카이브

2020-07-05 13:43:51

티스토리 블로그의 스킨 중 북클럽을 선택하셨다면 스킨의 오른쪽 하단에 전체 방문자수와 오늘과 어제 방문자 수가 몇 명인지 확인할 수 있는 카운터를 확인하셨을 겁니다. 

 

 이부분을 가리는 법에 대해서 알려드리겠습니다. 

 

 

 

적용 순서

 

1. 관리자 페이지로 이동합니다. 

 

2. 왼쪽 메뉴에서 [꾸미기] [스킨 편집]을 클릭합니다.

 

3. 상단에 있는 [html 편집] 버튼을 클릭합니다. 

 

4. html 편집 화면에서 732번 근처에 위치한 아래의 html 문구를 찾아주세요.

 

<!-- 전체 방문자 -->
<div class="count">
<h2>전체 방문자</h2>
<p class="total">[##_count_total_##]</p>
<p>Today : [##_count_today_##]</p>
<p>Yesterday : [##_count_yesterday_##]</p>
</div> 

 

5. 주석을 이용해서 다음과 같이 변경해 줍니다. 

 

<!-- 전체 방문자 
<div class="count"> 
<h2>전체 방문자</h2> 
<p class="total">[##_count_total_##]</p> 
<p>Today : [##_count_today_##]</p> 
<p>Yesterday : [##_count_yesterday_##]</p> 
</div> --> 

 

 

 

 

결과 

 

전체를 지우시는 것보다는 위와 같이 설정하시면 나중에 전체 방문자수를 다시 되돌릴 때 4. 와 같이 변경만 하면 되니까 편리합니다. 

 

 

블로그 페이지로 돌아가서 새로고침을 해보면 위와 같이 전체 방문자수와 오늘과 어제 방문자 수가 사라진 것을 확인할 수 있습니다. 

 

마무리

 

북클럽 스킨에서 방문자수를 가리는 방법에 대해서 알아봤습니다. 

티스토리 블로그는 각 블로그 스킨마다 카운터를 나타내는 html 태그가 조금씩 다릅니다. 

북클럽 이외의 다른 스킨을 사용하시고 방문자 카운터를 제외하시고 싶은 분도 있을 겁니다.

 

그런 분들은 위의 내용을 참고하셔서 다른 스킨에서도 적용하시면 됩니다. 

각 스킨의 html 편집창에서 [ctrl+f]를 누르고 list, total 혹은 today 등의 검색어로 찾아보시면 쉽게 파악할 수 있습니다.
