민생회복쿠폰 780만장 할인쿠폰 신청

개요

  • 대상: 국민 모두
  • 규모: 총 780만 장
  • 포맷: 카드 포인트, 모바일 쿠폰, 지역화폐
  • 배포 방식: 플랫폼별 앱/홈페이지 선착순 발급
  • 목적: 지역상권·숙박·문화 소비 진작

할인쿠폰별 혜택

분야 할인 금액 비고
숙박 1박당 2만~3만 원 할인 국내 호텔·펜션, 일부 OTA 연동
영화 최대 6천 원 할인 (1인 최대 4매) CGV·롯데시네마 등 주요지점 적용
공연·전시 3천~1만 원 할인 예매 시 자동 또는 수동 적용 가능

✔ 포인트: 1인당 쿠폰 수량 제한 있으므로 빠른 사용권 확보 필요

영화나 숙박은 전산과가 되어서 문제가 없지만, 공연/전시의 경우 횟수를 최대한 사용하기 위해서는 매번 다른 플랫폼을 이용해서 공연/전시 예매를 해야합니다.

지급 및 사용 일정

  • 지급 시작 예정보다 빠를 수 있음
  • 지급 방식: 지역화폐, 포인트, 모바일 쿠폰
  • 사용 시작: 발급 즉시
  • 사용 기한: 발급일로부터 ~4개월 이내

선착순 배포이므로 지급 시점 전에 미리 쿠폰 플랫폼 가입 및 알림 설정 권장

발급처 및 방법 - 민생회복쿠폰 780만장 할인쿠폰 신청

숙박 쿠폰

  • 플랫폼: 여기어때, 야놀자, 인터파크투어
  • 절차: 앱 접속 → ‘780만 할인쿠폰’ 팝업 확인 → ‘쿠폰 스탬프/발급’ 클릭 → 숙박 예약 시 사용

영화 쿠폰

  • 플랫폼: CGV, 메가박스, 롯데시네마
  • 절차: 앱 로그인 → 쿠폰 탭 내 ‘할인쿠폰’ 확인 → 예매 단계에서 적용

공연·전시 쿠폰

  • 플랫폼: 예스24, 인터파크 티켓
  • 절차: 공연/전시 선택 → 쿠폰 적용 가능 여부 확인 → 할인 적용 후 결제

유의사항

  1. 1인당 분야별 쿠폰 수량 제한 있으며, 조기 소진 주의
  2. 자동 적용되지 않는 경우도 있으므로 결제 화면에서 할인 체크 필수
  3. 플랫폼별 알림 설정 필수 – 쿠폰 오픈 즉시 알림 받을 수 있도록

마무리하며 - 민생회복쿠폰
