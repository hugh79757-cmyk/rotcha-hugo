KISA 보안패치 시스템이란?

개요

KISA는 ‘한국인터넷진흥원(Korea Internet & Security Agency)’의 약자로, 사이버 보안과 인터넷 안전을 관리하는 정부 산하 기관입니다. 이 기관은 실제로 다양한 보안 위협에 대응하기 위해 긴급 보안패치를 제공하곤 합니다.

하지만 최근 많은 사용자들이 접하는 ‘KISA 보안패치 시스템’은 공식 프로그램이 아닌 사칭 악성코드일 가능성이 매우 큽니다. 이 프로그램은 사용자의 명확한 동의 없이 설치되고, 자동으로 팝업을 띄우며, 광고 노출 및 추가 프로그램 설치를 유도하기 때문입니다.

악성코드와 공식 프로그램 구분하는 방법

설치 경로 Program Files 폴더 AppData, Temp 등 숨김 폴더
실행 방식 수동 실행 또는 단발성 실행 부팅 시마다 자동 실행, 반복적으로 팝업 발생
출처 kisa.or.kr, 보호나라 등 공식 도메인 알 수 없는 URL, 해외 IP
광고 포함 여부 없음 광고, 추가 앱 설치 유도
프로그램 이름 SecurityPatch, KISA 등 무작위 영문+숫자 이름, exe 확장자 포함

위 기준 중 하나라도 해당된다면, 이미 컴퓨터에 악성코드가 침투해 있을 가능성이 높습니다.

삭제 및 해결방법

① 정식 프로그램일 경우

  • 제어판 → 프로그램 제거 항목으로 진입
  • KISA, SecurityPatch 등 유사한 항목이 있다면 삭제
  • 시작 프로그램 (Ctrl + Shift + Esc → 시작 프로그램 탭) 비활성화
  • PC 재부팅 후 이상 유무 확인

② 악성코드일 경우

  • 백신 프로그램 정밀 검사
    V3, 알약, Windows Defender 등 최신 업데이트된 백신으로 시스템 전체 검사 실행
  • 전문 제거 도구 사용
    AdwCleaner, Malwarebytes 같은 악성코드 전용 툴로 추가 검사 및 제거
  • 자동 실행 항목 점검
    ‘작업 관리자’ 또는 ‘msconfig’에서 자동 실행되는 프로그램 중 의심 항목 해제
  • 브라우저 설정 초기화
    크롬, 엣지, 웨일 등 브라우저에서 불필요한 확장 프로그램 제거 및 설정 초기화
  • AppData, Temp 폴더 정리
    C:\Users\사용자명\AppData\Local\Temp 등에서 수상한 .exe 파일 삭제
  • 시스템 복원 또는 초기화(최후 수단)
    위 조치로도 해결되지 않을 경우, 윈도우 복원지점을 이용하거나 초기화를 고려

이런 경우 반드시 의심하세요

  • 매번 부팅 시 보안 패치 팝업이 뜬다
  • KISA라는 이름인데 광고창이 따라온다
  • 브라우저가 자동으로 특정 사이트를 열고, 게임 또는 금융 앱 설치를 유도한다
  • 알 수 없는 프로그램이 제어판에 등록돼 있다

예방을 위한 보안 수칙

  • 출처 불명의 이메일, 링크, 파일은 절대 열지 않기
  • 반드시 정품 윈도우와 정식 백신 사용하기
  • 토렌트, 크랙 프로그램 사용 시 주의
  • 보안 관련 알림은 반드시 공식 기관 홈페이지(kisa.or.kr)에서 재확인

마무리하며

최근 유포되는 ‘KISA 긴급 보안패치 시스템’ 팝업은 정식 보안 프로그램을 사칭한 악성코드일 가능성이 매우 높습니다. 반복적인 팝업, 광고 유도, 자동 실행 현상이 나타난다면 즉시 삭제 및 백신 점검을 진행해주시기 바랍니다. 컴퓨터 보안은 예방이 최우선이며, 의심되는 경우 절대 방치하지 마세요.

정확한 검사 도구와 조치를 통해 내 PC를 안전하게 지키는 것이 가장 중요합니다.
